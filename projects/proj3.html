Rocket Jumper is a challenging platformer where the user cannot jump, but instead propel yourself with the use of a rocket. The player must avoid various hazards and use the enviroment to platform around.

As the gameplay programmer, I implemented the core gameplay features, level design, and tile based level system.

  • Custom 2D Engine using C++
  • Implemented tile based level system
  • Designed and implemented core gameplay
  • Designed 6 levels
